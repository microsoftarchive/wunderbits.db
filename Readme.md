## How to use

    $ npm install wunderbits.db

## Develop and contribute

You need to have node and npm installed. Then fork this repo and open it in your terminal.

### Install dependencies

    $ make install

### Run tests

    $ make test

### Run JSLint

    $ make lint

### Build for distribution

    $ make build

### Publish to NPM

    $ make publish
